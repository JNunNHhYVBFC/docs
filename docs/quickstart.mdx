# Быстрый старт

Это руководство поможет вам начать работу с ботом-ассистентом Yandex Cloud за несколько простых шагов.

## Шаг 1: Подключение к боту

1. Откройте Telegram
2. Перейдите по ссылке: [@yc_assistant_bot](https://t.me/yc_assistant_bot)
3. Нажмите кнопку "Start" или отправьте команду `/start`

## Шаг 2: Авторизация

Для полноценной работы с ботом необходимо авторизоваться в Yandex Cloud:

1. Отправьте команду `/auth`
2. Следуйте инструкциям бота для авторизации
3. Подтвердите доступ к вашему аккаунту Yandex Cloud

## Шаг 3: Основные команды

После авторизации вам доступны следующие команды:

- `/help` - показать список всех команд
- `/services` - просмотр доступных сервисов
- `/price` - расчет стоимости услуг
- `/support` - получить помощь
- `/docs` - поиск по документации

## Примеры использования

### Выбор сервиса

```bash
/services
# Бот предложит выбрать категорию сервисов
# Следуйте подсказкам для выбора оптимального решения
```

### Расчет стоимости

```bash
/price
# Укажите интересующие вас сервисы
# Бот рассчитает примерную стоимость
```

### Поиск в документации

```bash
/docs compute instance
# Бот найдет релевантную информацию о виртуальных машинах
```
