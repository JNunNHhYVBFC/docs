# Сервисы Yandex Cloud

## Категории сервисов

### Compute

- **Compute Cloud** - Виртуальные машины для любых задач
- **Serverless** - Бессерверные вычисления
- **Containers** - Управление контейнерами
- **Functions** - Запуск кода без серверов

### Storage

- **Object Storage** - Хранение объектов любого размера
- **Managed Service for PostgreSQL** - Управляемые базы данных
- **Managed Service for MongoDB** - NoSQL базы данных
- **Managed Service for Redis** - In-memory хранилище

### Networking

- **Virtual Private Cloud** - Изолированные сети
- **Load Balancer** - Балансировка нагрузки
- **Cloud DNS** - Управление DNS-записями
- **Network Load Balancer** - Балансировка на L4

### AI/ML

- **DataSphere** - ML-разработка
- **Yandex GPT** - Языковые модели
- **Vision** - Компьютерное зрение
- **Speechkit** - Распознавание речи

## Как выбрать сервис

1. Используйте команду `/services` в боте
2. Опишите вашу задачу
3. Бот предложит оптимальные варианты

## Примеры использования

### Создание виртуальной машины

```bash
/services compute
# Выберите Compute Cloud
# Следуйте инструкциям для создания VM
```

### Настройка базы данных

```bash
/services database
# Выберите нужный тип БД
# Бот поможет с конфигурацией
```

## Расчет стоимости

Для расчета стоимости сервисов:

1. Используйте команду `/price`
2. Выберите интересующие сервисы
3. Укажите параметры (CPU, RAM, storage)
4. Получите расчет стоимости

## Оптимизация расходов

Бот поможет оптимизировать расходы:

- Анализ использования ресурсов
- Рекомендации по масштабированию
- Выбор оптимального тарифа
- Настройка автомасштабирования
